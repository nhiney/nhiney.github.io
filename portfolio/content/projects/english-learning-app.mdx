## Overview

A standalone Python application focused on providing a simple, distraction-free environment for learning English.

## Key Features

- **JSON Storage**: Lightweight data management ensures portability and easy backups.
- **Dynamic Quiz**: Self-testing mode for vocabulary reinforcement.
- **Search & Filter**: Quickly find words and categorize them by difficulty.

## Technical Stack

- **Language**: Python 3
- **GUI**: Tkinter (Standard Library)
- **Data**: JSON persistence
