## Project Overview

The Smart Medical Booking App is a mobile-first solution for patient-to-clinic interactions. It provides a user-friendly interface for booking appointments, managing medical history, and receiving real-time notifications.

## Key Features

- **Real-time Synchronization**: Powered by Firebase Firestore for instant updates on appointment status.
- **Secure Authentication**: Integration of Firebase Auth for patient and doctor logins.
- **Appointment Scheduling**: Intuitive calendar interface for selecting dates and time slots.
- **Medical Records Management**: Secure storage and retrieval of personal health documents.

## Tech Stack

- **Mobile Framework**: Flutter (Dart)
- **Backend-as-a-Service**: Firebase
- **Database**: Cloud Firestore (NoSQL)
- **Authentication**: Firebase Auth
