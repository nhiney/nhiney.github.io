SportField: Booking & Admin

March 20, 2024•1 min read

Overview

A robust web application for field owners and sports enthusiasts to manage and book fields easily.

Key Features

  • Automated Validation: Real-time checking of field availability during booking.
  • Admin Dashboard: Comprehensive management of users, fields, and revenue reports.
  • Dynamic Pricing: Support for different rates based on peak hours and field types.
  • Secure Auth: Laravel-based authentication for both users and administrators.

Technical Stack

  • Framework: Laravel 10
  • Database: MySQL with optimized indexing for time-range queries.
  • Templating: Blade / Tailwind CSS
