Sports Field Booking System

January 20, 2024•1 min read

Project Overview

This system allows sports facility owners to manage their fields and for users to book time slots conveniently. It includes automated confirmations and administrative tools for management.

Key Features

  • Administrative Dashboard: Complete control over fields, users, and bookings.
  • Automated Notifications: Email alerts for newly booked slots and cancellations.
  • User Booking: Intuitive selection of date, time, and specific fields.
  • Reporting: Basic statistics on facility usage and revenue.

Tech Stack

  • Framework: Laravel (PHP)
  • Database: MySQL
  • Frontend: Tailwind CSS, Alpine.js
  • Mail System: SMTP for automated notifications
